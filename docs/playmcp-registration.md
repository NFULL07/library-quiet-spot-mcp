## MCP 설명

```text
도서관 정보나루 공개 데이터를 활용해 도서관 이름만으로 이용 추이 데이터가 제공되는 도서관의 한산한 방문 시간대, 인기 도서의 소장 여부, 책 제목 또는 ISBN 기반 다음 독서 후보를 알려주는 MCP입니다. 도서관 코드를 몰라도 이름으로 검색하고, 대출·이용 분석 데이터를 가공해 바로 판단할 수 있는 형태로 제공합니다.
```

정독도서관 한산한 시간 알려줘
